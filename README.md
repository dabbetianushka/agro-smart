# agro-smart
agro-smart is a web application which provides farmers to predict the crop yield and recommends sutaible fertilizers and pesticides based on crop ,soil and weather
conditions .
## Problem Statement
In agriculture, many farmers still depend on guesswork or traditional methods to decide which crop to grow and which fertilizer or pesticide to use.
Because of this, they often face low crop yield, higher farming costs, and damage caused by pests and diseases.
There is a need for a simple and easy-to-use system that can guide farmers with basic recommendations
## System Architecture
User enters details → Web interface processes the input → Logic provides prediction or recommendation → Result is displayed on the screen
## Tech Stack
- HTML – for creating the structure of the website
- CSS – for styling and layout
- JavaScript – for interactivity and displaying results
## Setup Instructions
1. Download the repository
2. Open the index.html file in any web browser
3. Enter the required inputs and view the results
